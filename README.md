## 📢 프로젝트 개요
웨스트민스터 대요리문답 학습을 위해 기존 Class형 React로 제작된 WLC-Service를 함수형 React로 마이그레이션 하며 디자인 및 새로운 기능들을 추가하여 개선하였습니다. <br/>
실제로 이전 서비스를 사용하며 개인적으로 느낀점과 주변의 피드백을 취합하여 학습 흐름과 데이터 정합성을 중심으로 전반적인 UX를 개선하는데 집중하였습니다.

<br/>

## 🔧 주요 개선 사항
### 🎨 디자인 전면 개선 및 다크/라이트 테마 기능 추가 : <br/>
- 전체 UI를 재설계하여 가독성과 사용성을 개선
- 사용 환경에 따라 선택 가능한 다크/라이트 테마 전환 기능을 구현
<br/>

### 💾 최근 본 문답 기능 추가 :   <br/>
- Local Storage를 활용하여 사용자별 최근 열람한 문답을 최대 3개까지 저장
- 재방문 시 이전 학습 흐름을 자연스럽게 이어갈 수 있도록 UX 개선
<br/>

### 📚 각주 모음 페이지 추가 :   <br/>
- 문답에 해당하는 구절만 제공되던 성경 각주를 한 페이지에서 모아볼 수 있도록 구성
- 문답별로만 제공되었떤 성경 근거 구절을 
<br/>

### 🔗 각주별 해당 문답으로 연결되는 매칭 기능 추가 :   <br/>
- 각주 리스트에서 문답 번호 클릭 시 해당 문답으로 즉시 이동 가능하도록 연결
- 성경 구절 → 문답으로 자연스럽게 이어지도록 하여 학습 흐름 강화
<br/>

### 🔎 각주 모음 페이지 및 문답 리스트 페이지에 검색 기능 추가 :   <br/>
- 키워드 기반 검색 기능을 도입하여 문답 번호, 내용, 성경 구절을 빠르게 탐색 가능
- 검색을 통해 학습 효율 증대
<br/>

### 🧵 한글/영문 각주 데이터의 정합성을 맞춰 더욱 정확한 내용 제공 :   <br/>
- 한글/영문 각주 데이터를 비교·정리하여 내용 불일치 문제 해결
- 언어에 관계없이 동일한 학습 경험을 제공하도록 데이터 구조 개선
